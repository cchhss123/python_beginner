# Python 初學者教材

本專案旨在提供一份簡明易懂的 Python 初學者教材，涵蓋了從環境建立到基礎語法的各個方面。

## 目錄

* [單元 1：Python 開發環境建立與第一支程式](unit1_setup_and_hello.md)
* [單元 2：資料型態與變數](unit2_data_types_and_variables.md)
* [單元 3：判斷式與迴圈](unit3_conditionals_and_loops.md)
* [單元 4：串列與元組](unit4_lists_and_tuples.md)
* [單元 5：字典與集合](unit5_dictionaries_and_sets.md)

## 簡介

本教材由淺入深，逐步引導初學者掌握 Python 的基礎知識。每個單元都包含清晰的說明和實用的範例程式碼，幫助學習者快速上手。

* **單元 1：Python 開發環境建立與第一支程式**
    * 介紹如何安裝 Python 環境和程式碼編輯器。
    * 引導學習者撰寫並執行第一支 Python 程式。

* **單元 2：資料型態與變數**
    * 講解 Python 中常用的基礎資料型態（整數、浮點數、字串、布林值）。
    * 介紹變數的概念、命名規則和使用方法。

* **單元 3：判斷式與迴圈**
    * 講解如何使用 `if`、`elif`、`else` 進行條件判斷。
    * 介紹 `for` 和 `while` 迴圈的使用方法，以及如何控制迴圈的流程。

* **單元 4：串列與元組**
    * 深入介紹串列（list）和元組（tuple）這兩種重要的序列型資料結構。
    * 講解它們的特性、常用操作和適用場景。

* **單元 5：字典與集合**
    * 介紹字典（dictionary）這種鍵值對的資料結構。
    * 講解集合（set）這種不重複元素的資料結構以及其常用操作和集合運算。

## 如何使用本教材

您可以直接瀏覽本專案中的 Markdown 文件 (`.md` 檔案) 來學習每個單元的內容。每個單元通常包含以下部分：

* **標題：** 清晰地標示單元主題。
* **簡介：** 簡要介紹本單元的學習目標。
* **內容說明：** 詳細講解相關概念和語法。
* **範例程式碼：** 提供可直接執行的程式碼範例，幫助理解。

建議您搭配 Python 環境和程式碼編輯器，親手執行範例程式碼，並嘗試修改和擴展它們，以加深理解。
